RESEARCH SECTION

Comparing EcoMart’s design with popular e-commerce websites.

Research Objective

The main objective of this research is to study how different e-commerce websites design their layouts and user interfaces. Local platforms like Daraz and global websites such as Amazon, Apple, and EcoMart Malaysia are compared to see how EcoMart can improve its design.

Reference Website Component Investigated EcoMart’s Adaptation
EcoMart Malaysia
(ecomart.com.my)		 Branding and Header Navigation EcoMart (Nepal) follows a similar eco-friendly design but keeps the navigation simpler and easier to understand.
Daraz Nepal
(daraz.com.np)		 Individual Product Page and Checkout Page EcoMart takes inspiration from Daraz’s product detail layout and checkout flow while avoiding excessive ads and banners.
Amazon India
(amazon.in)		 Product Card Structure EcoMart uses Amazon’s product card layout, clearly showing the product image, name, price, and action button.
Apple
(apple.com)		 Landing Page Design and CTA Placement EcoMart is inspired by Apple’s landing pages by using clear primary and secondary call-to-action buttons.

Design Comparisons

This section shows how EcoMart (Nepal) adapts design elements from popular reference websites. Each comparison displays the original reference on the left and EcoMart’s implementation on the right.

EcoMart Malaysia Branding

EcoMart Malaysia uses a consistent green color palette and simple header layout for eco-friendly branding. EcoMart (Nepal) adopts these elements but simplifies the navigation to improve usability while maintaining the brand identity.

Apple Landing Page

Apple’s landing page emphasizes large visuals and clear primary and secondary buttons. EcoMart (Nepal) uses this approach by keeping the main actions clear and the landing page clean, improving user engagement.

Amazon Product Cards

Amazon product cards show the image, title, price, and action buttons clearly. EcoMart (Nepal) follows this structure to make products easy to browse and purchase.

Daraz Product and Checkout Flow

Daraz provides detailed product info and a structured checkout flow. EcoMart (Nepal) adapts this layout to make shopping smooth while keeping the interface less cluttered and user-friendly.
